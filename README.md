# Generieke diensten voor digitale infrastructuur 

Dit project geeft antwoord op de vraag: [Welke generieke vertrouwensdiensten zijn er nodig voor online zakendoen?](https://alkem.io/tip/collaboration/welkegeneriekevert-1149).

Zie voor een beschrijving van de aanpak: [Inventarisatie generieke vertrouwensdiensten versie 0.5 (9 juli 2024)](https://alkem.io/tip/collaboration/welkegeneriekevert-1149/posts/inventarisatiegeneriekeve/dashboard).

Binnen dit project gebruiken we twee termen voor generieke diensten voor digitale infrastructuur:

- Een trust service is een elektronische dienst die normaliter tegen vergoeding wordt verleend, en die door een EU-lidstaat wordt erkend als vertrouwensdienst.
- Een generic trusted ecosystem service is een vertrouwens- of infrastructuurdienst die kan worden toegepast op informatie-uitwisseling in verschillende sectoren.
